## Custom Prompts in Console  
_August 20, 2024_

Custom Prompts with metadata were introduced to provide enterprises with 
better control over how prompts are processed and integrated into the 
generative pipeline. This is particularly useful for creating tailored RAG 
applications.

**More information:**: [Custom Prompts in Console](#)

## User Defined Function Reranking
_August 15, 2024_

Vectara introduced User Defined Function Reranking, giving enterprises more 
granular control over search result ordering by applying custom ranking logic, 
such as sorting by price, date, or any custom metadata. This flexibility is 
particularly useful for e-commerce and enterprise search applications.

**More information:**: [User Defined Function Reranking](#)

## Mockingbird LLM  
_July 16, 2024_

Vectara's purpose-built LLM for Retrieval-Augmented Generation (RAG) enhances 
generative outputs with reliable citations and a focus on multilingual 
support. This model is optimized for enterprises aiming to deliver grounded, 
trustworthy AI responses.

**More information:**: [Mockingbird LLM](#)

## Vectara API v2  
_June 6, 2024_

The Vectara API v2 provides a more RESTful, intuitive structure with simpler 
authentication, new top-level objects, and better defaults for hybrid search 
and reranking, making it easier to develop applications with Vectara’s GenAI 
platform.

New API Endpoint(s): [Vectara API v2 Overview](#)

**More information:**: [Vectara API v2 Documentation](#)

## Vectara Multilingual Reranker v1 (Slingshot)  
_May 28, 2024_

The state-of-the-art Vectara Multilingual Reranker, also known as Slingshot, 
provides more accurate neural ranking than the initial Boomerang retrieval. 
While computationally more expensive, it offers improved text scoring across a 
wide range of languages, making it suitable for diverse content. Slingshot 
boosts enterprise search quality, especially in globally distributed 
organizations.

**More information:**: [Multilingual Reranker](#)

## Generative Response Styling  
_May 15, 2024_

With this release, Vectara allows users to format citations in summaries using 
Markdown or HTML. This feature is essential for integrating generative 
responses into web-based applications and ensuring citations are clear and 
appropriately formatted.

**More information:**: [Generative Response Styling](#)
